## API

| Prop       | Type      | Description                     |
| ---------- | --------- | ------------------------------- |
| `label`    | `string`  | Text label.                     |
| `filled`   | `boolean` | Uses filled visual treatment.   |
| `outlined` | `boolean` | Uses outlined visual treatment. |
| `tonal`    | `boolean` | Uses tonal visual treatment.    |
