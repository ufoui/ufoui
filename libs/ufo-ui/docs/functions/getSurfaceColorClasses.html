      Object with uui-text-*, uui-border-*, and uui-bg-* class names for base and on-color.

      Remarks

      Uses the inverseColorMap to find the related “on” color (e.g. 'surfaceContainer' → 'onSurface'). Produces text, border, and background variants for both. All names are kebab-cased (e.g. 'surfaceContainerHigh' → 'surface-container-high').

      Example

      getSurfaceColorClasses('surfaceContainer');
      // → { textColor: 'uui-text-surface-container', textOnColor: 'uui-text-on-surface', ... }
